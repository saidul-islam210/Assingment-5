Question-3: What is event delegation in the DOM?
Event delegation is a very interesting mehod in JavaScript where instead of adding event listeners to multiple child elements,we add a single event listener to a parent element.
Question-4: What is event bubbling in the DOM?
Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM. It allows parent elements to respond to events triggered by their child elements.
